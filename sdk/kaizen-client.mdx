## `KaizenClient`

This is the core asynchronous client for the Kaizen SDK.

### Initialization
```python
from kaizen import KaizenClient
client = KaizenClient.from_env()
```

### Configuration
The client automatically reads environment variables:
- `KAIZEN_BASE_URL`
- `KAIZEN_API_KEY`
- `KAIZEN_TIMEOUT`

You can also pass in a custom config:
```python
from kaizen import KaizenClient, KaizenClientConfig

config = KaizenClientConfig(api_key="sk-123", timeout=60)
client = KaizenClient(config=config)
```

### Context Manager Support
```python
async with KaizenClient.from_env() as client:
    ...
```
