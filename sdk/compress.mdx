---

## `client.compress(payload)`

Compress a full JSON payload using the Kaizen optimization engine.

### Parameters
- `payload` (`EncodeRequest` or `dict`): Input request containing the model, messages, etc.

### Returns
Compressed dict suitable for upstream usage.

### Example
```python
compressed = await client.compress({
    "model": "gpt-4",
    "messages": [{"role": "user", "content": "Hello!"}]
})
```
